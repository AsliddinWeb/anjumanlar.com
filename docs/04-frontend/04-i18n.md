# i18n — 3 til (uz/ru/en)

## Strategiya

- **Default locale:** `uz` (O'zbekcha)
- **Qo'shimcha:** `ru` (Русский), `en` (English)
- **URL strukturasi:** `prefix_except_default`
  - `/` → uz
  - `/ru/` → ru
  - `/en/` → en
- **Browser language detection** yoqilgan
- **Cookie:** `i18n_redirected` saqlanadi

## Konfiguratsiya

`nuxt.config.ts` da modul allaqachon sozlangan. Tarjima fayllari `locales/` papkasida:

```
frontend/locales/
├── uz.json
├── ru.json
└── en.json
```

## Backend'dan keladigan multilingual matnlar

DB da ko'p tilli maydonlar JSONB sifatida saqlanadi:

```json
{
  "title": {
    "uz": "Tarix metodologiyasi",
    "ru": "Методология истории",
    "en": "Methodology of History"
  }
}
```

Komponentda:

```vue
<script setup lang="ts">
const { locale } = useI18n()

const displayTitle = computed(() =>
  props.book.title[locale.value] || props.book.title.uz
)
</script>
```

Yoki composable yarating:

```typescript
// composables/useLocalized.ts
export const useLocalized = () => {
  const { locale } = useI18n()
  
  return (field: Record<string, string> | undefined) => {
    if (!field) return ''
    return field[locale.value] || field.uz || Object.values(field)[0] || ''
  }
}
```

Foydalanish:

```vue
<script setup>
const localized = useLocalized()
</script>

<template>
  <h1>{{ localized(book.title) }}</h1>
</template>
```

## SEO bilan birga

`nuxt.config.ts` da i18n SEO yoqilgan. Har sahifaga hreflang teglari avtomatik qo'shiladi:

```html
<link rel="alternate" hreflang="uz" href="https://monografiya.com/books" />
<link rel="alternate" hreflang="ru" href="https://monografiya.com/ru/books" />
<link rel="alternate" hreflang="en" href="https://monografiya.com/en/books" />
<link rel="alternate" hreflang="x-default" href="https://monografiya.com/books" />
```
